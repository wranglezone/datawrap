# datawrap

Helpers for the final steps of dataset preparation: auto-cleaning,
creating data dictionaries, and documenting package datasets.

## Installation

You can install the development version of datawrap from
[GitHub](https://github.com/) with:

``` r
# install.packages("remotes")
remotes::install_github("wranglezone/datawrap")
```
